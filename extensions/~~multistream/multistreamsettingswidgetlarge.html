Multistream Settings

Multistream using StreamRoller as the multiplexer to take OBS stream and output to different platforms
Notes:

  • ffmpeg is required to run, if you have a version installed (check the command prompt if not sure. enter ffmpeg -version' to see if you have one already installed. If in stalled select 'use installed ffmpeg box below', else you can select 'use StreamRoller ffmpeg' which will install a version of ffmpeg into the extensions bin folder for you)
  • StreamRoller server needs starting before OBS. To start the server use the enable/disable checkbox in the small settings page (on the Home/livePortal page)
  • Any updates to this page will only take effect when next starting a stream
  • For Twitch testing (without going live) set 'Additional Params ' to '?bandwidthtest=true' (without quotes)
  • For YouTube testing (without going live) start the stream on YouTube using 'Unlisted' rather than 'Public'
  • For YouTube backup server (for some reason mine failed a lot with the standard server) set 'Additional URL ' to '?backup=1' (without quotes)
