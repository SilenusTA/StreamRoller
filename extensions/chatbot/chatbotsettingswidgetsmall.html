Chatbot Settings

Chatbot is a ChatGPT powered chatbot that can be asked questions or just hang out in chat talking with users.

Note: to set the login credentials go to the bottom of the main settings page for chatbot (tab at top of main page)

Profile in use: chatbotprofileselectedname

Activity level (how often the bot responds)

This setting will change how long the bot pauses in between responses.
low = 8-10 minutes
med = 4-5 minutes
high = 0 minutes
(These values can be fine tuned in the main settings page)

Message count per request

This is how many messages to send to the AI bot. ie 5 means collect the next 5 messages (after the timeout above)and send them to get a response
