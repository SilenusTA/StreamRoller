OBS Settings

Default Scenes Name

These are the OBS scene names that will be attached to the obs controls on the main page. We use items in fron of OBS names to be able to group them into Main, Secondary and 'everything' (no identifies needed) else scenes so we can quickly switch between these groups (settings are in the obs settings link) Default setings are '##' to indicate a main scense (change the scene name in OBS to start with ##) and ** for secondary scenes.

OBS Scene buttons

Extra scene buttons to display on top of defaults start, live, intermission and end buttons (set identifier tags in the obs extension settings first)

Twitter
On startup you can post a Tweet to let people know you are live. There three parts to the tweet.
  1. Main message
  2. Extra message
  3. Random fact (If enabled)
Streamroller will calculate the length of the tweet and do the following options in order depending on size
  1. If there is enough space, post "main message" + "Extra Message" + "Random fact"
  2. If 1. doesn't fit then try "Main message" + "Random Fact"
  3. If 2. doesn't fit then try "Main Message" + "Extra Message"
