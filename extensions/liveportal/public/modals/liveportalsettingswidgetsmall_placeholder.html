Windows Text To Speech

This setting will play out chat using windows TTS.
You can change the output device (ie if you just chat in your headset and not on stream).
To change the output for Windows text-to-speech go to Windows Search Bar
Search "control panel" -> "Speech Recognition" -> "Advanced Speech Options" -> "Text To Speech" -> "Advanced". Note this only seems to work for windows default voices, installed voices are 'non local' and don't seem to use this setting for the audio output.
You can then select the device you want the TTS to go out on.
